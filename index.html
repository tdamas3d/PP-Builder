About PP Builder

PP Builder is a web-based application developed in HTML and Javascript.
PP Builder provides users with the possibilities to search and build products pages, for only in-stock products, from major e-commerce platforms such as Amazon, eBay, and Alibaba. The PP Builder has a simple interface for users with two main input fields. The first in put field allows users to type in the url of the e-commerce platform, for example, Amazon, that the products are searched from. The second input field is for the search keywords, that is, the name or category of products, for example, computers.
The PP Builder works as follows:

  1. Enter the url of the site to search products from
  2. Enter the keywords, in this case, the name or type of products
  3. Search products and Generate PP (products pages or catalog)

Automatically creates products page for the latest products on amazon or any other eCommerce platform in any niches
